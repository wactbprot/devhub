response

(response conf task)

Gets and returns a stub response if one is registered in :stub-response-file.

Example:

(response (u/config) {:TaskName "VS_SE3-get-valves-pos"})

select-response

(select-response kw rs mode)

Selects the response depending on the configuration. Implemented methods are:

  • :rand (default)
  • :first (fallback)
