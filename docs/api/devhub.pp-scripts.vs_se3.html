check

(check rs m)

Returns a vector of maps.

Example:

(check [89] {:ThingA [0 0] :ThingB [0 1]})
;; =>
;; [{:ThingA true} {:ThingB false}]

valves

(valves task {rs :_x})

Returns exchange structures like

{ToExchange:
  {E7_open:    {Bool: true},
   E3_open:    {Bool: true},
   E15_open:   {Bool: true},
   E10_open:   {Bool: false},
   E20_open:   {Bool: false},
   E18_open:   {Bool: true},
   E10_closed: {Bool: true},
   ...
   registers [89, 0, -102, 0, -102, 0, 21, 0, -91]}}

Example:

(valves {} {:_x [85 0 16 0 16 0 21 0 7]})
