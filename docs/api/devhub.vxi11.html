handler

(handler {conf :vxi} task)

Handles VXI11 queries.

Example: ```clojure (def vc (u/config)) (handler vc {:Device “gpib0,8” :Host “e75465” :Value “:meas:func”})

(handler vc {:Wait 10 :Repeat 3 :Device “inst0” :Host “e75495” :Value “ch101() ”})

;; => ;; {:data ;; {:_x [MEASURING 1.015], ;; :_t_start [1607513740001], ;; :_t_stop [1607513740028], ;; :_dt [27]}} ```
