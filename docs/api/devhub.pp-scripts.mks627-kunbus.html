kunbus-vec

(kunbus-vec v)

Returns a vector of bytes that can be turned into a float.

Example:

(kunbus-vec  [0 0 0 16128 2836 120 0 0])
;; =>
;; [63 20 0 0] ; 0.578125 mbar
