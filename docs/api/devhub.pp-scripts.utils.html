calc-seq

(calc-seq v o)

Same as operable-seq but returns a vector of numbers.

Example:

  (calc-seq ["1" 1.234E-5 0    "a"  :number]
              [true  true     true false  false])  
;; =>
;; [1 1.234E-5 0]

operable

(operable v)

Returns a vector of booleans indicating if the values are usable in calculations.

Example:

(operable ["1" 1.234E-5 0 "a" :number])
;; =>
;; [true true true false false]
(operable 1)
;; =>
;; nil
view source

operable-seq

(operable-seq v o)

Shrinks the vector v down to operable values depending on vector o.

Example:

(operable-seq ["1" 1.234E-5 0    "a"  :number]
              [true  true     true false  false])  
;; =>
;; ["1" 1.234E-5 0]

stdev

(stdev v)

Calculates the standard deviation of the vector v.

Example:

(stdev [0 1 2])
;; =>
;; 1.0

(stdev [0])
;; =>
;; nil

(stdev [])
;; =>
;; nil

(stdev nil)
;; =>
;; nil
