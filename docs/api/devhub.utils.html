map-vals

(map-vals f m)

Map f over every value of m. Returns a map with the same keys as m, where each of its values is now the result of applying f to them one by one. f is a function of one arg, which will be called which each value of m, and should return the new value. Faster then map-vals-transient on small maps (8 elements and under)

(meas-vec coll)

Returns data if data is a map. Transforms data from a single measurement to a measurement vector if the length is greater than 1.
